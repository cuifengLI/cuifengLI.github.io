什么是hexo

官方解释：快速简洁且高效的博客框架，hexo使用Markdown(或其他渲染引擎)解析文章，在几秒内，即可生成你满意的静态网页（个人博客网站）

以上的博客是运行到本地的，如何将自己的博客网页发布到公网，让大家都能看到你的博客？下面就需要我们借助Github Pages功能将自己建立的博客发布到公网。

搭桥到github

进行这部分的前提就是你需要有一个属于自己的github账号，没有的话直接上github官网按照指示要求注册属于自己的账号即可。以下步骤默认你有github账号进行操作。

  • 点击头像左侧加号按钮第一个“new repository”功能创建一个代码仓库。

  • 仓库名按照要求应该是.github.io，建议直接勾选 Initalize this repository a README按钮，然后点击Create repository创建代码仓库即可。

  • 创建完成之后，点击进入Settings，查看Github Pages是否选中并启用，保证启用即可。

  • 然后使用 hexo-deployer-git 插件进行部署(切记)

  • 进入_config.yml文件夹，将编译后的静态部署至master分支，其中具体格式如下：
